# BookStore FrontEnd React


### Set port
.env
```
PORT=8081
```

## Project setup

In the project directory, you can run:

```
npm install
# or
yarn install
```

or

### Compiles

```
npm start
# or
yarn start
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.



# BookStore Backend Nodejs


### Set port
.env
```
PORT=8080
```

## Project setup

In the project directory, you can run:

```
npm install
# or
yarn install
```


### Run 
```
node index.js
```

Service will be active at [http://localhost:8080]
